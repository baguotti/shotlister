# Shotlister

A progressive web app (PWA) for managing shot lists and schedules.

## Development

Install dependencies:
```bash
npm install
```

Start the development server:
```bash
npm run dev
```

## Production Build

Build the static application (PWA) for production:
```bash
npm run build
```

The output will be generated in the `dist/` directory. The service worker will cache these assets for full offline support.
